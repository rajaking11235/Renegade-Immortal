STEP 1 — Learn the Basics (1–2 Weeks)*

The 4 things in the beginning:

 1. Unreal Engine Interface

* Viewport
* Outliner
* Content Browser
* Blueprints (Visual Coding)
* Details Panel

2. Basic Objects

* Static Mesh
* Character
* Camera
* Lighting
* Landscape

3. Blueprint Programming
Blueprints are enough to build a full game.

You will learn:

* Movement
* Jump
* Sprint
* Camera rotation
* Interactions
* Health bar

4. Importing Assets

From:

* Blender (models)
* Material Maker (textures)
